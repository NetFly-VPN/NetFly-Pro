## App核心功能:
##### 1. 登录ssmgr
##### 2. 获取 节点, 流量等用户信息
##### 3. 商城: 获取网站商品列表
##### 4. 支付: 集成支付宝支付功能, 用户可以直接在app内部完成支付下单功能( 注意: 因为我的网站没有配置支付宝回调, 所以app完成支付后要过几分钟才能获取支付到支付成功. 当配置支付宝回调接口后, 支付成功后几秒钟后App就可以获取支付成功的消息了 )
##### 5. 注册新用户: 用户可以在app内部完成注册账户功能

## 可定制范围:
1. 自定义App名称, 可与网站同名
2. 自定义App图标.
3. 提供App 安装链接:(包括iOS 安装链接, 不需要切换Apple ID, 国内直接可以下载安装)
